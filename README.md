# Requisitos
feh, python, pip, venv

# Instalar
Após clonar o repo, basta executar os seguintes comandos
```bash
pip install -r requirements.txt
pyinstaller --onefile main.py -n manga-tupi
sudo cp dist/manga-tupi /usr/local/bin
```
# Usar
```bash
manga-tupi
```
